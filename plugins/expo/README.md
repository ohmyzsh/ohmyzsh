# expo plugin

This `expo` plugin sets up aliases for the [expo](https://expo.io) React-Native cli

To use it, add `expo` to the plugins array of your zshrc file:

```zsh
plugins=(... expo)
```

## Aliases

| Alias | Command |
| --- | --- |
| es | expo start |
| ei | expo install |
| eba | expo build:android |
| ebi | expo build:ios |
| el | expo login |
| eh | expo help |
| eu | expo upgrade |
