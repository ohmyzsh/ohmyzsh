# wails

Full alias list for Wails CLI.

To use it, add `wails` to the plugins array in your zshrc file:

```zsh
plugins=(... wails)
```

## Requirements

[Just the wails cli and is dependencies.](https://wails.io/docs/gettingstarted/installation)

## Aliases

| Alias | Command |
| --- | --- | 
| wb | wails build  |
| wd | wails dev |
| wdoc | wails doctor |
| wi | wails init |
| wu | wails update |
| ws | wails show |
| wg | wails generate |
| wv | wails version |
| wh | wails -help |
