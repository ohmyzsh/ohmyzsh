# Tekton CLI

This plugin provides autocompletion for [tkn](https://tekton.dev/docs/cli/). A command for easier interaction with Tekton CI/CD components.

To use it, add `tkn` to the plugins array of your zshrc file:

```bash
plugins=(... tkn)
```
