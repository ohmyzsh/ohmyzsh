# snap plugin

This plugin sets up aliases for the common [snap](https://snapcraft.io/docs/getting-started) commands

## Aliases

| Alias | Full command |
| --- | ---|
| sv | snap version |
| sf | snap find |
| si | snap install |
| sin | snap info |
| sr | snap remove |
| sref | snap refresh |
| srev | snap revert |
| sl | snap list |
| sd | snap disable |
| se | snap enable |
