# flatpak plugin

This plugin sets up aliases for the common [flatpak](https://flatpak.org/) commands

## Aliases

| Alias | Full command |
| --- | ---|
| fpv | flatpak --version |
| fpf | flatpak search |
| fpi | flatpak install |
| fpiu | flatpak install --user |
| fpin | flatpak info |
| fpr | flatpak uninstall |
| fpref | flatpak update |
| fpl | flatpak list |
| fpd | flatpak disable |
| fpe | flatpak enable |
| fprl | flatpak remote-list |
| fpra | flatpak remote-add --if-not-exists |
