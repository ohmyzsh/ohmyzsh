# flatpak plugin

This plugin sets up aliases for the common [flatpak](https://flatpak.org/) commands

## Aliases

| Alias | Full command |
| --- | ---|
| fpv | flatpak --version |
| fpf | flatpak search |
| fpi | flatpak install |
| fpin | flatpak info |
| fpr | flatpak uninstall |
| fpref | flatpak update |
| fpd | flatpak disable |
| fpe | flatpak enable |
