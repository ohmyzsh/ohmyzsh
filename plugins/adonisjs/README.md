# Laravel

This plugin adds aliases for [AdonisJS Framework](https://adonisjs.com/).

```
plugins=(... adonisjs)
```

## Development

| Alias | Description |
|:-:|:-:|
| `ads`  | `adonis serve --dev`  |
| `adrepl`  | `adonis repl` |
| `adinstall`  | `adonis install` |
| `adt`  | `adonis test` |
| `adaddon`  | `adonis addon` |
