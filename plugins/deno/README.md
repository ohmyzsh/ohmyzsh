# Deno Plugin

This plugin sets up completion and aliases for [Deno](https://deno.land).

## Aliases

| Alias | Full command        |
| ----- | ------------------- |
| db    | deno bundle         |
| dc    | deno compile        |
| dca   | deno cache          |
| dfmt  | deno fmt            |
| dh    | deno help           |
| dli   | deno lint           |
| drn   | deno run            |
| drA   | deno run -A         |
| drw   | deno run --watch    |
| dru   | deno run --unstable |
| dts   | deno test           |
| dup   | deno upgrade        |
